AI Output Validation Service

A lightweight validation service for ensuring AI outputs conform to expected schemas

What is the AI Output Validation Service?

The AI Output Validation Service is a dedicated API built with FastAPI and Pydantic v2 that validates outputs from AI agents against predefined schemas. It ensures that AI-generated content meets your expected structure and requirements before it flows through your automation pipelines.

Key Features

Schema Validation

Validate AI outputs against predefined schemas using Pydantic v2's powerful validation capabilities.

Multiple Validation Types

Support for various output types including generic outputs, recommendations, summaries, and classifications.

Detailed Error Reporting

Get comprehensive error messages that pinpoint exactly what's wrong with your AI output.

n8n Integration

Seamlessly integrate with n8n workflows to validate AI outputs before further processing.

API Key Authentication

Secure your validation endpoints with API key authentication.

Fast Performance

Built with FastAPI and Pydantic v2 for high-performance validation.

How It Works

1

Generate output with an AI agent

Create content with any AI system such as OpenAI's GPT, Anthropic's Claude, or your custom AI solution.

2

Send the output to our validation endpoint

Make a POST request to /validate/{validation_type} with your AI output in the request body.

3

Receive validation results

Get back a standardized response indicating whether the output is valid or invalid, with detailed error messages if validation fails.

4

Continue your workflow with confidence

Process the validated data in your automation pipelines knowing it conforms to your expected structure.

Supported Validation Types

Generic

Basic validation for any AI output with a response text and optional confidence score.

Recommendation

Validates recommendation-style outputs with a list of recommendations and optional user context.

Summary

Validates text summary outputs with original text, summary, and optional key points.

Classification

Validates classification outputs with text, categories, and optional probabilities.

API Integration Example

Here's a simple example of how to use the validation service in your application:

import requests

# Your AI output data
ai_output = {
    "response_text": "This is a response from the AI",
    "confidence_score": 0.92
}

# Send to validation service
response = requests.post(
    "https://dkogc88csc8kk4cg0o8s48ss.selmanayan.com/validate/generic",
    headers={"x-api-key": "your_api_key"},
    json=ai_output
)

# Check validation result
if response.json()["status"] == "valid":
    # Process the validated data
    validated_data = response.json()["validated_data"]
    print("Validation successful:", validated_data)
else:
    # Handle validation errors
    errors = response.json()["errors"]
    print("Validation failed:", errors)

n8n Integration

Easily integrate with n8n workflows to validate AI outputs before further processing:

  1. Set up an AI agent node in n8n (e.g., OpenAI)
  2. Add an HTTP Request node:
    • Method: POST
    • URL: https://dkogc88csc8kk4cg0o8s48ss.selmanayan.com/validate/{validation_type}
    • Headers: x-api-key: your_api_key
    • Body: Output from the AI agent node
  3. Add a Switch node to handle the validation result:
    • Valid case: {{$node["HTTP Request"].json["status"] === "valid"}}
    • Invalid case: {{$node["HTTP Request"].json["status"] === "invalid"}}
